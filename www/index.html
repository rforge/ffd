R-Package: FFD - Freedom From Disease using Two-Stage Sampling

Introduction

In practice, when conducting surveys to substantiate freedom from disease in large populations two-stage sampling strategies are often used in order to account for herd-level clustering of diseases. With two-stage sampling the questions that need to be addressed are a) how many herds need to be sampled and b) how many animals per herd need to be tested. Using a modified hypergeometric formula the optimal sample sizes can elegantly be computed, while incorporating imperfect diagnostic tests and finite populations; see Cameron and Baldock (1998a,b).

In the package FFD, tools for calculating optimal sample sizes (on animal and herd level) using sampling strategies "individual sampling" or "limited sampling" (see Ziller, Selhorst, Teuffert, Kramer, and Schlüter (2002)) are implemented. The idea behind individual sampling is that the number of animals to test is computed for each herd size in order to guarantee a certain herd sensitivity independent of the herd size. With limited sampling a pre-fixed number of animals is tested for each herd (=sample limit), irrespective of the herd size. Using FFD cost optimal sampling strategies can be determined finding the optimal herd sensitivity or sample limit, respectively.

The package provides S4-classes and methods for convenient handling, as well as conventional functions in order to guarantee maximum flexibility.

Features

  • Calculation of sample sizes for strategies "individual sampling" and "limited sampling"
  • Diagnostic tools for the optimization of efficiency and costs
  • A-posteriori calculation of the alpha-error corresponding to a specific sample
  • Sampling methods using either a fixed number of herds or dynamically updating the achieved significance of the sample and randomly sampling herds until a specified significance level is met

Example

A simple example using simulated data of 15287 sheep holdings. For survey parameters design prevalence = 0.2%, significance level = 5%, intra herd prevalence = 20%, sensitivity of the diagnostic test = 90% and cost per tested animal = 7 euro, cost per tested herd = 30 euro (excluding the costs for the diagnostic tests) diagnostics are run in order to find the cost-optimal sample size for limited sampling. > ## Load library and read data
> library("FFD")
> data(sheepData)

> ## Specify the survey parameters:
> mySurveyData <- surveyData(nAnimalVec = sheepData$nSheep,
+    populationData = sheepData, designPrevalence = 0.002,
+    alpha = 0.05, intraHerdPrevalence = 0.2,
+    diagSensitivity = 0.9, costHerd = 30, costAnimal = 7)

> ## Diagnostics for limited sampling:
> myLtdSampleSummary <- ltdSamplingSummary(survey.Data = mySurvey,
+    sampleSizeLtdMax = 30)
> plot(myLtdSampleSummary)

Package Installation

To install this package directly within R type: 

install.packages("FFD", repos="http://R-Forge.R-project.org")

or download source code or binary from R-Forge.
